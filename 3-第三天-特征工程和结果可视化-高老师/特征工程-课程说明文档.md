# 决策树课程-练习说明文档

## 1. 环境要求

- 1.1 依赖库

#### 安装软件
vscode最新版
Anaconda3-4.2
python3.5
matplotlib == 1.5.3
numpy == 1.13.3
pandas == 0.18.1
scikit_learn == 0.19.1

#### 电脑要求
x64位，最低4GB内存以上（建议8GB以上），Windows或Mac或Linux均可。
